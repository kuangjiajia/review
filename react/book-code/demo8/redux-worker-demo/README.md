# Redux Worker Demo

Redux worker demo

## 运行
第一步，请先安装node（建议版本8.9.4）

第二步，切换到 demo/nqueen 或 demo/sortTable，安装依赖

```bash
$ npm install
```

如果安装很慢，可以试试使用淘宝的镜像，速度会快很多

```bash
$ npm install --registry=https://registry.npm.taobao.org
```

第三步，进行构建

```bash
$ npm run watch
```

第四步，打开 http://localhost:8080/webpack-dev-server/ ，预览效果
